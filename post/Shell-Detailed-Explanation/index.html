Shell简介#

Shell-外壳，他与kernel-内核相对应

只要是能给用户提供命令行的程序，都可以称之为shell

bash是现在最常用的shell（我用zsh

Shell也提供了各种小工具，比如grep，sed，awk等等，供用户方便使用

Shell思考#

感觉shell跟git有一定相似之处？ 两者都通过命令-参数的方式来操作系统(所谓unix哲学)

Shell工具#

awk#

基本用法#

1

awk '{print $0}' file.txt

$0表示整行，$1表示第一个字段，$2表示第二个字段以此类推

1
2

$ echo 'this is a test' | awk '{print $0}'
this is a test

1
2

$ echo 'this is a test' | awk '{print $3}'
a //识别到第三个字段是a

内置变量#

  • NF: Number of Fields（字段数量）
1
2

$ echo 'this is a test' | awk '{print $NF}'
test

NF-1就表示倒数第二个字段

  • NR: Number of Records（行数）
1
2
3
4
5
6

$ awk -F ':' '{print NR ") " $1}' demo.txt
1) root
2) daemon
3) bin
4) sys
5) sync

NR表示行数
